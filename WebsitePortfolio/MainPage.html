Past Projects

Project 1

Our team designed a MySQL Workbench database for a simulated electronic store, efficiently managing employees, customers, inventory, purchase history, and membership discounts. We also explored the option for customers to make purchases online or in-store. Employing an Entity-Relationship diagram, we established various relationships, including many-to-many and one-to-many connections. By testing numerous queries, we illustrated diverse purchase scenarios within the system.

Project 2

A completed senior graduation project using Unreal Engine resulted in a 3D space shooter game inspired by Starfox 64 and other classics. Players control a spaceship near Earth, collecting coins for a high score while avoiding asteroids and enemy fire. Although the game is still in development as a personal project, a demo presentation is available on a private YouTube link, with plans for a future release on Steam.

Project 3

In my upper division classes, I actively contributed to the open-source project Blender, gaining valuable experience in configuration management, version control, build systems, and project documentation. Through cloning the repository, using Git commands, and employing CMake GUI, I honed my skills in managing large software projects. Additionally, I studied the flow of control for different mesh operators using the debug configuration build of Blender and our preferred IDE. Lastly, I conducted unit testing of fundamental Blender vector operations using Python scripts and the Python unit test framework.
